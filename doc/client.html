    Declaration

    class IrcClient;

    A client used for sending and reading data to/from an IRC server. Automatically tracks users and channels.

    • onNickInUse

      Declaration

      bool delegate(in string oldNick)[] onNickInUse;

      Raised when this client's nick name is in use. If the event is not handled by any delegates, an exception is thrown.

      Discussion

      To handle the exception, you must set a new name by changing nickName, and then return true.

      Parameters

      oldNick

      The nick name that is already in use.

      Return Value

      true if it has been handled, else false. If the event is not handled by any delegates, an exception is thrown.

      See Also

      nickName
